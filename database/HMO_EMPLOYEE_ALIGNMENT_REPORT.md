## Alignment Strategy

### Provider Selection
- All providers are genuine top 5 HMO companies in the Philippines
- Realistic network sizes, member counts, and establishments dates
- Appropriate employer/employee contribution rates

### Plan Selection
- Each provider's **best plan** was selected as the primary plan
- Mix of Standard, Premium, and Executive tiers
- Annual premiums range from ₱15,000 to ₱55,000

### Employee Matching
- Senior managers → Premium/Executive plans
- Regular employees → Standard/Premium plans
- Consideration given to department and responsibility level

### Dependent Inclusion
- All dependents from HR Core schema included
- Spouse relationships honored
- Children added with birth dates

## How to Use

### To Import into Database:
```bash
mysql -u root public_html < database/seed_hmo_data.sql
```

### To Verify Data:
```sql
-- Check provider counts
SELECT COUNT(*) as total_providers, provider_name 
FROM hmo_providers 
GROUP BY provider_name;

-- Check employee enrollments
SELECT e.employee_id, e.first_name, hp.plan_name, p.provider_name 
FROM employees e
JOIN employee_hmo_enrollments ehe ON e.employee_id = ehe.employee_id
JOIN hmo_plans hp ON ehe.plan_id = hp.id
JOIN hmo_providers p ON ehe.provider_id = p.id;

-- Check claims sum
SELECT SUM(claim_amount) as total_claims, 
       COUNT(*) as claim_count 
FROM hmo_claims;
```
